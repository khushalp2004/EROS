# EROS Enhancement Roadmap

## Phase 1 (Implemented)
- Environment-based configuration and secret removal
- API rate limiting for sensitive routes
- Backend unit tests scaffold
- Frontend unit tests scaffold
- CI pipeline (GitHub Actions)

## Phase 2 (Next)
- Audit logging model + middleware
- Incident timeline replay API and UI playback controls
- Smart dispatch scoring engine with explainable ranking

## Phase 3
- Analytics dashboard (dispatch/response KPIs and heatmaps)
- Offline queueing and sync for location/emergency updates
- Accessibility audit fixes (keyboard support, aria labels, contrast)

## Phase 4
- Production deployment setup and monitoring
- API documentation (OpenAPI/Swagger)
- Architecture and ER diagrams
